# This folder holds question images (diagrams, figures, etc.)
# for Logical Reasoning and other image-based questions.
#
# Supported formats: .png, .jpg, .jpeg, .gif, .webp, .svg
#
# Naming convention:
#   {question_id}_{description}.png
#   Examples: lr002_series.png, lr002_optionA.png, lr003_figure.png
#
# After adding images here, run:
#   node src/db/uploadQuestionImages.mjs
# to upload them to Firebase Storage and generate image-url-map.json
