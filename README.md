# MCAverse - The Ultimate MCA Entrance Preparation Platform

![MCAverse Logo](/public/mcaverse-logo.png)

**MCAverse** is a comprehensive, open-source educational platform designed to help students prepare for MCA entrance exams (like NIMCET). Built with **Next.js 15**, **TypeScript**, and **Firebase**, it offers a robust suite of tools including mock tests, daily practice problems, video lectures, and a community forum.
